# Spinner

Spinner component with different props.

#### Example

<Canvas of={Spinner.Default} />

## Usage

```ts
import {Spinner} from "bagheera-ui";

const Example = () => {
  return (
      <Spinner
        size="small"
      />
  );
};

export default Example;
```
